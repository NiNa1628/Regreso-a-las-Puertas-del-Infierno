# 🥋 JUDOMEX – Plataforma Web para Gestión Deportiva

JUDOMEX es una aplicación web dinámica desarrollada para apoyar la enseñanza, práctica y organización de actividades relacionadas con el judo. La plataforma permite a estudiantes, entrenadores y organizadores interactuar mediante herramientas diseñadas para facilitar la gestión deportiva.

## ✨ Características principales

* Registro e inicio de sesión de usuarios.
* Autenticación segura mediante Firebase Authentication.
* Inicio de sesión con correo electrónico y Google Sign-In.
* Creación y edición de perfiles de usuario.
* Inscripción a competencias y torneos.
* Carrito de compras para artículos deportivos.
* Navegación entre las diferentes secciones de la plataforma.
* Panel administrativo para consulta de registros y participantes.
* Gestión y sincronización de información en tiempo real mediante Firebase Firestore.

## 🛠️ Tecnologías utilizadas

* PHP
* Firebase Authentication
* Firebase Firestore
* HTML5
* CSS3
* JavaScript

## 🎯 Objetivo del proyecto

Desarrollar una solución tecnológica que contribuya a la difusión y organización del judo, ofreciendo herramientas digitales que mejoren la experiencia de estudiantes, entrenadores y organizadores de eventos deportivos.

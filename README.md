# 🪖 Regreso a las Puertas del Infierno – Videojuego Shooter 2D

Regreso a las Puertas del Infierno es un videojuego shooter en dos dimensiones desarrollado en Java bajo el paradigma de Programación Orientada a Objetos (POO). El proyecto se inspira en la cruda realidad del conflicto contra el narcotráfico en México, ofreciendo una experiencia de acción táctica ambientada en las regiones áridas del norte del país.

## ✨ Características principales

* **Ambientación realista:** Escenarios detallados que recrean ciudades del Norte de México, caracterizadas por climas secos y entornos urbanos conflictivos.
* **Sistema de bandos en conflicto:** Enfrentamiento directo entre las fuerzas del Ejército Mexicano (EjeMex) y facciones del Cártel de Sinaloa (CDS).
* **Mecánicas dinámicas de combate:** Inspirado en la jugabilidad rápida y letal de títulos emblemáticos como *Wolfenstein* y *Hotline Miami*.
* **Trabajo en equipo:** El jugador no lucha solo; cuenta con un escuadrón táctico de apoyo aliado controlado por inteligencia artificial para cumplir los operativos.
* **Gestión de entidades mediante POO:** Estructura modular para el control independiente de personajes, proyectiles, estados de salud, inventarios de munición y obstáculos reactivos.
* **Sistema de colisiones y física 2D:** Implementación avanzada para la detección de impactos, líneas de visión de los enemigos y delimitación de coberturas en el mapa.
* **Despliegue multiplataforma:** Arquitectura compatible con ejecución local y simulación web interactiva.

## 🛠️ Tecnologías utilizadas

* **Java** (Lógica principal del juego y aplicación del paradigma POO)
* **Greenfoot** (Framework de desarrollo para simulaciones y entornos interactivos 2D)
* **Programación Orientada a Objetos** (Uso riguroso de Herencia, Polimorfismo y Encapsulamiento)

## 🎯 Objetivo del proyecto

Desarrollar un videojuego interactivo que rinda homenaje a los pioneros del género shooter, utilizando herramientas de programación orientada a objetos en Java para modelar un sistema complejo de combate en tiempo real, ambientado en un contexto histórico y geográfico contemporáneo de alta relevancia.

## 📚 Aprendizajes obtenidos

Durante el desarrollo de este proyecto fortalecí mis conocimientos en:
* **Estructuras de datos y algoritmia en tiempo real:** Control de flujos continuos mediante ciclos de juego (*game loops*).
* **Programación Orientada a Objetos Avanzada:** Modelado de entidades complejas mediante herencia abstracta y métodos polimórficos para actores del entorno.
* **Gestión de física básica en 2D:** Manejo de coordenadas cartesianas, vectores de movimiento y detección geométrica de colisiones.
* **Inteligencia Artificial para videojuegos:** Diseño de patrones de búsqueda, patrullaje y ataque autónomo para los enemigos.
* **Diseño de niveles y optimización:** Estructuración de mapas balanceados que integran de forma fluida la dificultad, la narrativa visual y los obstáculos del entorno.
